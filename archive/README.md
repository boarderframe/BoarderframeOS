# Archive Directory

This directory contains old scripts, temporary fixes, and deprecated code that may be useful for reference but are no longer actively used.

## Directory Structure

- **`debug/`** - Debug scripts used during development
- **`demos/`** - Demo and proof-of-concept scripts
- **`fixes/`** - One-time fix scripts for specific issues
- **`old/`** - Deprecated implementations
- **`old_ui/`** - Previous UI implementations
- **`scripts/`** - Miscellaneous one-off scripts
- **`test_html/`** - HTML test files
- **`tests/`** - Old test scripts (new tests should go in `/tests`)

## Purpose

These files are kept for:
1. **Historical Reference** - Understanding how issues were fixed
2. **Code Examples** - Seeing different implementation approaches
3. **Debugging** - May contain useful debugging techniques
4. **Recovery** - In case we need to revert or reference old code

## Guidelines

- **DO NOT** run these scripts without understanding their purpose
- **DO NOT** import or depend on code in this directory
- **DO** feel free to reference these for learning
- **DO** consider deleting very old files periodically

## Note

If you find yourself needing code from here regularly, consider:
1. Moving it to the active codebase
2. Rewriting it as a proper utility
3. Documenting why it's needed

This is an archive - treat it as a museum, not a toolbox!
