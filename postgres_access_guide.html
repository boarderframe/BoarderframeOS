🗄️ PostgreSQL Database Access Guide

BoarderframeOS Database Connection Information & Tools

🔌 Connection Details

Host: localhost
Port: 5434 (custom port, not default 5432)
Database: boarderframeos
Username: boarderframe
Password: boarderframe_secure_2025
Container: boarderframeos_postgres

Connection URL:

postgresql://boarderframe:boarderframe_secure_2025@localhost:5434/boarderframeos

🖥️ PostgreSQL Control Panel Options

1. pgAdmin 4 (Recommended GUI)

The most popular PostgreSQL management tool with a web interface.

Download pgAdmin

Quick Setup:

  1. Install pgAdmin 4
  2. Add New Server with these settings:
    • Name: BoarderframeOS
    • Host: localhost
    • Port: 5434
    • Database: boarderframeos
    • Username: boarderframe
    • Password: boarderframe_secure_2025

2. TablePlus (Premium Option)

Modern, native database management tool for macOS, Windows, and Linux.

Get TablePlus

Connection String for TablePlus:

postgresql://boarderframe:boarderframe_secure_2025@localhost:5434/boarderframeos?sslmode=disable

3. DBeaver (Free & Open Source)

Universal database tool supporting PostgreSQL and many other databases.

Download DBeaver

4. Adminer (Lightweight Web UI)

Simple PHP-based database management in a single file.

Quick Docker Setup:

docker run -d -p 8080:8080 --network boarderframeos_default adminer

Access at: http://localhost:8080

🔧 Command Line Access

Using Docker:

docker exec -it boarderframeos_postgres psql -U boarderframe -d boarderframeos

Using psql directly:

PGPASSWORD=boarderframe_secure_2025 psql -h localhost -p 5434 -U boarderframe -d boarderframeos

Useful psql commands:

\dt # List all tables \d table_name # Describe table structure \l # List all databases \du # List all users/roles \q # Quit psql

📊 BoarderframeOS Specific Tables

Core Tables:

  • agents - Agent registry and configuration
  • departments - Department structure
  • divisions - Division organization
  • leaders - Leadership registry
  • llm_cost_tracking - LLM API usage and costs
  • llm_budget_tracking - Budget limits per agent
  • analytics_events - System analytics
  • payments - Payment records
  • customers - Customer information

View LLM Costs Example:

SELECT agent_name, model_used, SUM(total_cost) as total_cost, COUNT(*) as requests FROM llm_cost_tracking GROUP BY agent_name, model_used ORDER BY total_cost DESC;

⚠️ Important Notes:

  • The PostgreSQL server runs on port 5434, not the default 5432
  • Make sure Docker is running before attempting to connect
  • The database is containerized in Docker as boarderframeos_postgres
  • For production use, consider changing the default password

🚀 Quick Start

The easiest way to get started is to use pgAdmin 4 or TablePlus with the connection details above.

For command-line access, use the Docker exec command provided.
