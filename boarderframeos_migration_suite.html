1 Pre-Migration Check

pre_migration_check.py

Verifies your source BoarderframeOS installation is complete before migration. Checks for all critical files, validates configurations, and generates a readiness report.

python pre_migration_check.py /path/to/current/boarderframeos
Core Files Check
Verifies all system files
Agent Validation
Checks agent implementations
Database Inventory
Lists all database files

2 Enhanced Migration

enhanced_migrate_config.py

Comprehensive migration script that copies all BoarderframeOS components. Includes all files from CLAUDE.md, handles sensitive data, and creates migration guides.

# Dry run first python enhanced_migrate_config.py /source /target --dry-run # Actual migration python enhanced_migrate_config.py /source /target
Complete Coverage
All components from CLAUDE.md
Sensitive Data
Smart .env handling
Database Scripts

3 Post-Migration Setup

post_migration_setup.py

Automated setup script for the new environment. Creates virtual environment, installs dependencies, configures Docker services, and runs database migrations.

⚠️ Important Notes

  • Database files (.db) must be migrated manually for security
  • API keys and secrets need manual configuration in .env
  • Ensure Python 3.13+ is installed on the target system
  • Docker Desktop is required for PostgreSQL and Redis
  • Review all generated reports before proceeding

📋 Quick Start Guide

Step 1: Check Source

cd /current/boarderframeos python pre_migration_check.py .

Step 2: Migrate Files

python enhanced_migrate_config.py /current/boarderframeos /new/location

Step 3: Setup New Environment

cd /new/location python post_migration_setup.py

Step 4: Migrate Databases

./migrate_databases.sh /current/boarderframeos

Step 5: Configure Environment

cp .env.secret .env # Edit .env and add your ANTHROPIC_API_KEY and other secrets nano .env

Step 6: Verify Installation

python migration_verification_enhanced.py .

Step 7: Start BoarderframeOS

source venv/bin/activate # or venv\Scripts\activate on Windows python startup.py
