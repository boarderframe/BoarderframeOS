Claude Code Specific Configurations

Files to Migrate

.claude/settings.json - Main Claude Code settings
.claude/settings.local.json - Local overrides and permissions
.mcp.json - MCP server configurations
.env - Environment variables (including API keys)

MCP Servers to Configure

boarderframeos-filesystem - File system access
boarderframeos-memory - Knowledge graph persistence
Custom BoarderframeOS MCP servers (ports 8000-8011)
Important: After migration, you'll need to reconfigure Claude Code on the target system:
  1. Install Claude CLI from https://claude.ai/download
  2. Copy .claude/ directory to target system
  3. Run 'claude mcp install' to set up MCP servers
  4. Verify environment variables are set correctly

Migration Success Criteria

Your migration is successful when all of the following are verified:
All verification scripts pass without critical errors
PostgreSQL and Redis containers are running
Corporate HQ is accessible at http://localhost:8888
MCP servers respond to health checks
Agents can be imported and initialized
Claude Code can connect to MCP servers
Message bus is operational
No missing critical files reported

Troubleshooting Guide

Common Issues and Solutions

!
PostgreSQL connection failed: Ensure Docker is running and check port 5434
!
MCP servers not starting: Verify Node.js is installed and .mcp.json is valid
!
Import errors: Run 'pip install -r requirements.txt' in virtual environment
!
Claude Code not connecting: Check .claude/settings.json permissions
