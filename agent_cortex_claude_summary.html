💰 How Costs Are Calculated

1. Model Selection by Agent Tier

Executive Tier (Solomon, David)

Primary: Claude 3 Opus

Cost: $0.015 input / $0.075 output per 1K tokens

Quality: 0.95 (highest)

Department Heads

Primary: Claude 3.5 Sonnet

Cost: $0.003 input / $0.015 output per 1K tokens

Quality: 0.93

Specialists

Primary: Claude 3 Sonnet

Cost: $0.003 input / $0.015 output per 1K tokens

Quality: 0.88

Workers

Primary: Claude 3 Haiku

Cost: $0.00025 input / $0.00125 output per 1K tokens

Quality: 0.82

2. Cost Calculation Formula

Input Cost = (Input Tokens / 1000) × Input Price per 1K Output Cost = (Output Tokens / 1000) × Output Price per 1K Total Cost = Input Cost + Output Cost

3. Example Calculation

Component Claude 3 Opus Claude 3 Haiku
Input Tokens 500 500
Output Tokens 200 200
Input Cost 500/1000 × $0.015 = $0.0075 500/1000 × $0.00025 = $0.000125
Output Cost 200/1000 × $0.075 = $0.015 200/1000 × $0.00125 = $0.00025
Total Cost $0.0225 $0.000375

🔧 Current Implementation

How It Works:

  1. Agent makes request to http://localhost:8005/generate
  2. Agent Cortex determines agent tier (Executive, Department, Specialist, Worker)
  3. Selects appropriate Claude model based on tier and task complexity
  4. Makes API call to Anthropic with selected model
  5. Tracks cost based on estimated tokens
  6. Returns response with cost tracking information

Key Files:

  • /mcp/agent_cortex_server.py - API server on port 8005
  • /core/agent_cortex.py - Intelligent model selection
  • /core/agent_cortex_claude_config.py - Claude model registry
  • /core/cortex_client.py - Client wrapper for agents
  • /core/base_agent.py - Updated to use CortexClient

📈 Cost Optimization Features

  • Intelligent Model Selection: Uses cheaper models when quality requirements allow
  • Budget Controls: Can throttle agents exceeding daily/monthly limits
  • Fallback Strategies: Automatically switches to budget models if needed
  • Request Batching: Future enhancement to reduce API calls
  • Response Caching: Future enhancement to reuse common responses
