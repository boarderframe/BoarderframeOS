# 🚀 BoarderframeOS Quick Start

## Start the Dashboard

You can now start the BoarderframeOS dashboard with any of these simple commands:

### Option 1: Main Launcher
```bash
cd /Users/cosburn/BoarderframeOS
./start
```

### Option 2: UI Command
```bash
cd /Users/cosburn/BoarderframeOS
./ui
```

### Option 3: Dashboard Command
```bash
cd /Users/cosburn/BoarderframeOS
./dashboard
```

## What Happens

- 🌐 **UI server starts** on port 8888
- 🚀 **Browser opens** automatically to the dashboard
- 📊 **Dashboard shows** system status and setup instructions
- 💬 **Solomon interface** ready (offline until backend runs)

## Dashboard URL

**http://localhost:8888**

## Next Steps (Shown in UI)

1. **Run Setup**: `python setup_boarderframeos.py`
2. **Start Backend Servers** 
3. **Initialize AI Agents**

## Stop the Server

Press **Ctrl+C** in the terminal to stop the UI server.
