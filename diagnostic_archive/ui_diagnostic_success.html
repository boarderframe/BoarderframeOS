🔧 Available Endpoints

UI Diagnostics Endpoint

GET /api/ui/diagnostics

Provides comprehensive UI state information including:

  • ACC (Agent Communication Center) display status and configuration
  • Agent Cortex display status and configuration
  • Navigation structure and active tabs
  • Server counts and health status
  • Welcome tab metrics

            


            

                
Component State Endpoint

                
GET /api/ui/component/{component}

                
Query specific UI components. Available components:

                
    
                    
  • header - Logo, status text, and server counts
    • 
                    
  • servers_tab - Server categories and ACC entry details
    • 
                    
  • chat_integration - Chat button visibility and ACC integration
    • 
                

                
                

            


            

                
UI Change Preview Endpoint

                
POST /api/ui/preview

                
Preview UI changes before applying them. Send JSON with change details.
